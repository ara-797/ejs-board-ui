# ejs-board-ui
### [업무대비] EJS 연습용 프로젝트

1. EJS 설치하기
    - 전역 설치가 아닌 프로젝트 단위로 설치
    - Express와 함께 설치

    ```javascript
    // 설치
    npm install express ejs
    
    // 확인
    npm list ejs
    ```

2. 최소 서버 파일 만들기
    - app.js

3. 서버 실행
    ```javascript
    node app.js
    ```

<hr />

### SCSS 사용하기
```javascript
// 설치
npm install --save-dev sass

// 컴파일 명령어
sass assets/scss:assets/css --watch
```
